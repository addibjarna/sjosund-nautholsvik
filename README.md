# Sjósund Nauthólsvík – v21 text layout

Breytingar:
- Textinn „öryggi“ tekinn út úr undirfyrirsögn.
- Flóðagagnalínan fyrir ofan línurit er nú í tveimur línum:

```text
Flóðagögn Reykjavík: miðvikudagur 1. júlí 2026
Núna 00:17 • 0,88 m
```

Keyrsla:
1. Tvísmella á `start-windows.bat`
2. Opna `http://localhost:3050`
